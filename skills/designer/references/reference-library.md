# Reference Library

Use this file to choose the dominant aesthetic thesis for the project.

## Direction selector

| Direction | Best for | Signature traits | Risks |
|-----------|----------|-------------------|-------|
| Editorial / Magazine | Docs, content, brand stories | Strong hierarchy, asymmetric rhythm, generous reading flow | Can feel too quiet if contrast is weak |
| Luxury / Refined | Premium services, trust-heavy products | Controlled whitespace, elegant typography, restrained motion | Can become sterile without texture |
| Industrial / Utilitarian | Dev tools, ops, dashboards | Dense structure, visible frames, functional clarity | Can become too severe if softened poorly |
| Bento Grid / Modular | SaaS homepages, tool hubs | Mixed card sizes, modular tension, clear priority | Can become default if all cards are equal |
| Linear-like Minimalist | Developer products, consoles | Thin borders, compressed type, crisp spacing, precise hierarchy | Can flatten into bland minimalism |
| Brutalist | Creative brands, sharp propositions | Raw edges, high contrast, blunt hierarchy | Can become hostile if unreadable |
| Retro-futuristic | Creative tools, energetic products | Grids, chrome, glow, digital motion | Can become novelty if overused |
| Futuristic Glassmorphic | Analytics, futuristic apps | Depth, blur, layered panels, luminous accents | Can lose clarity if overdecorated |
| Organic / Natural | Wellness, sustainability, food | Soft curves, warm tone, tactile surfaces | Can become generic if too pastel |
| Playful / Toy-like | Education, consumer apps | Rounded shapes, bright accents, lively motion | Can become childish if unbounded |

## Selection rule

Pick one dominant direction based on the product's emotional job:
- does the product need trust and authority?
- does it need energy and forward motion?
- does it need reading comfort and editorial clarity?
- does it need density and utilitarian precision?

Only one direction should lead. Others can appear as supporting influences, but never as equal partners.
