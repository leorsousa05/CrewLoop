# Case Study Template

Use this as the narrative skeleton for every design spec.

## Problem

What user job is this interface solving?

## Audience

Who is using it, and what state are they in when they arrive?

## Insight

What single observation justifies the chosen visual direction?

## Solution

One sentence that ties the design direction to the product promise.

## Direction statement

Describe the chosen aesthetic in 2-3 sentences. Explain the emotional effect and why it fits the product.

## Proof points

- Reference files used.
- Key layout decisions.
- Key type decisions.
- Key color decisions.
- Key motion decisions.
