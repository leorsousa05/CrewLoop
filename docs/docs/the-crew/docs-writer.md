# Docs-Writer

**Phase:** Documentation

The Docs-Writer produces clear, actionable documentation tailored to the project type and audience.

## When to use

- Create or rewrite READMEs.
- Document modules, features, or capabilities.
- Pure documentation tasks with no code changes.
